# js 中一些意想不到的使用技巧（持续更新）

### 1. 箭头函数 `=>` 返回 `map` 对象

```
// 一般的写法
const makeMap = () => {
  return {key: 'value'};
};

// 完美的写法
const makeMap = () => ({key: 'value'});
```

### 2. 对象属性动态命名

对象属性名不确定，需要动态的传入。

```
// 一般的写法
const makeMap = (key, value) => {
  const obj = {};
  obj[key] = value;
  return obj;
};

// 完美的写法
const makeMap = (key, value) => ({[key]: value});
```
